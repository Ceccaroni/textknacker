# Leichte Sprache und Einfache Sprache – Italienisch (IT-CH)

> Wissensdatei für einen spezialisierten GPT zur Textumformulierung in Leichte Sprache und Einfache Sprache (Italienisch, Schweizer Kontext).  
> Stand: 21. Februar 2026

---

## 1. Begriffsklärung

### Leichte Sprache (Niveau A1–A2 GER)
- **Lingua facile** – der geläufigste Kurzbegriff für leicht verständliche Sprache im Italienischen (Quelle: EBGB/UFPD Leitfaden v1.1, Juni 2025; Perego, EASIT-Projekt).
- **Lingua facile da leggere** – «leicht zu lesende Sprache» (Quelle: Perego 2020, EASIT; Anffas).
- **Lingua facile da leggere e da capire** – Vollform, analog zum FALC (Quelle: Inclusion Europe; EBGB).
- **Lingua facile da capire** – «leicht zu verstehende Sprache» (Quelle: Perego, EASIT-Transcript).

> **Achtung:** Im Italienischen besteht **terminologische Uneinheitlichkeit**. Es gibt keinen einheitlich festgelegten Fachbegriff. Die Varianten werden teils austauschbar verwendet (Quelle: EBGB/UFPD Leitfaden v1.1; Perego 2020).

### Einfache Sprache (Niveau B1 GER)
- **Linguaggio chiaro** – der gebräuchlichste Begriff für «klare Sprache» / Plain Language (Quelle: EBGB/UFPD; Perego EASIT).
- **Linguaggio chiaro e semplice** – «klare und einfache Sprache» (Quelle: Perego EASIT-Transcript).
- **Semplificazione linguistica** – «Sprachvereinfachung», akademischer Terminus (Quelle: Perego EASIT).
- **Linguaggio semplice** – informeller (Quelle: Perego EASIT).
- **Plain Language** – Lehnwort, gelegentlich im akademischen Kontext (Quelle: Perego EASIT).

### Unterschied
Wie im Französischen wird im italienischen Sprachraum **nicht systematisch** zwischen Lingua facile (Easy Language) und Linguaggio chiaro (Plain Language) unterschieden. Die Regeln werden «oft vermischt und austauschbar verwendet» (Perego, EASIT 2021). Das EBGB unterscheidet jedoch klar zwischen A1/A2-Niveau (Lingua facile) und B1-Niveau (Linguaggio chiaro).

| | Lingua facile (A1–A2) | Linguaggio chiaro (B1) |
|---|---|---|
| Niveau GER | A1–A2 | B1 |
| Zielpublikum | Kognitive Beeinträchtigungen, schwere Leseschwäche | Breite Bevölkerung |
| Regelstrenge | Streng | Flexibler |
| Prüfgruppe | Empfohlen | Nicht obligatorisch |

(Quelle: EBGB/UFPD Leitfaden v1.1; Perego 2020; Guida Lingua facile MeteoSvizzera 2023)

### Existierende Regelwerke
- **Inclusion Europe «Information for all»** (2009): Europäische Standards, verfügbar in Italienisch.
- **EBGB/UFPD «Lingua facile – Guida per le amministrazioni pubbliche»** v1.0 (Juli 2023) / v1.1 (Juni 2025): Minimale Kriterien.
- **Anffas Linee Guida** (Pathways 2): Italienische Richtlinien für leicht verständliche Information.
- ⚠️ **Es gibt derzeit keine vollständigen, eigenen Richtlinien** für Lingua facile, die spezifisch für die italienische Sprache entwickelt wurden (Perego, EASIT 2021).
- Empfehlungen finden sich verstreut bei Cortellazzo & Pellegrino (2002), Fortis (2003), Sciumbata (2017).

## 3. Regelset – Leichte Sprache (Lingua facile, Niveau A1–A2)

Quellen: Inclusion Europe (2009); EBGB/UFPD Leitfaden v1.1 (2025); Guida Lingua facile MeteoSvizzera (2023); Perego/Sciumbata EASIT (2021).

### Wortschatz
| Regel | Quelle |
|---|---|
| Nur gebräuchliche, konkrete Wörter verwenden (z. B. «ricordare» statt «rammentare»). | Perego EASIT; IE 6 |
| Keine Fremdwörter, Latinismen oder Fachbegriffe. Falls nötig: sofort erklären. | IE 7, 11; EBGB |
| Keine Metaphern, Euphemismen (z. B. «morire» statt «venire a mancare»). | Perego EASIT; IE 10 |
| Einfache Verben statt Umschreibungen (z. B. «assicurare» statt «provvedere alla copertura assicurativa»). | Perego EASIT |
| Dasselbe Wort für dieselbe Sache verwenden. | IE 9; EBGB |
| Keine Abkürzungen oder Akronyme. Falls nötig: erklären. | IE 12; EBGB |

### Satzbau
| Regel | Quelle |
|---|---|
| Kurze Sätze; nur Hauptsätze (Soggetto – Verbo – Complemento). | IE 14; EBGB |
| Maximal eine Aussage pro Satz. | IE 19 |
| Jeder Satz auf einer neuen Zeile. | IE 17; EBGB |
| Kein Konjunktiv (Congiuntivo) – Indikativ verwenden. | Perego EASIT; EBGB |
| Kein Passiv – Aktiv verwenden. | EBGB; Perego EASIT |
| Kein Partizip Präsens oder Partizip Perfekt in Nebensatzkonstruktionen – explizite Formen bevorzugen. | Perego EASIT |
| Keine unpersönlichen Formen (z. B. «si deve» → «voi dovete»). | Perego EASIT |
| Keine doppelte Verneinung (z. B. «non infelice» → «felice»). | Perego EASIT; IE 16 |
| Direkte Ansprache verwenden. | IE 15; EBGB |
| Bejahende Formulierungen bevorzugen. | IE 16; EBGB |

### Textstruktur
| Regel | Quelle |
|---|---|
| Klare Überschriften und Zwischentitel. | IE 20; EBGB |
| Logische Reihenfolge der Informationen. | IE 18 |
| Aufzählungen mit Aufzählungszeichen. | IE 26 |
| Text linksbündig; kein Blocksatz. | IE 28; EBGB |
| Serifenlose Schrift, mindestens 14 pt. | IE 5–6 |
| Fettdruck für Hervorhebungen; keine Kursivschrift. | EBGB; IE |
| Bilder/Piktogramme verwenden (erwachsenengerecht). | IE 34–38 |
| Genügend Weissraum; nicht zu viel Text pro Seite. | IE 29–30 |
| Easy-to-Read-Logo verwenden. | IE 40 |

### Zahlen und Zeitangaben
| Regel | Quelle |
|---|---|
| Zahlen als Ziffern schreiben. | IE 42; EBGB |
| Keine römischen Ziffern. | IE 42 |
| Keine Prozentzahlen; «pochi» / «molti» verwenden. | IE 13; EBGB |
| Daten vollständig ausschreiben: «martedì 13 ottobre 2026». | IE 45 |
| Telefonnummern mit Leerzeichen gliedern. | EBGB |

### Typische Schwierigkeiten des Italienischen
| Schwierigkeit | Empfehlung | Quelle |
|---|---|---|
| Congiuntivo (Konjunktiv) – im Italienischen sehr häufig | Durch Indikativ ersetzen (auch wenn umgangssprachlich). | Perego EASIT |
| Participio presente/passato in Nebensatzkonstruktionen | Explizite Formen verwenden (z. B. «che ha fatto» statt «avente fatto»). | Perego EASIT |
| Forme impersonali (man-Konstruktionen) | Direkte Ansprache verwenden. | Perego EASIT |
| Latinismi | Durch italienische Alltagswörter ersetzen. | Perego EASIT |
| «d eufonica» (ed, od statt e, o) | Vermeiden: «e ora» statt «ed ora». | Perego EASIT |
| Doppelte Verneinung (typisch für Italienisch) | Eine einzige Aussage in bejahender Form. | Perego EASIT |
| Euphemismen | Direkte, klare Wörter verwenden. | Perego EASIT |
| Komplexe Periphrasen | Einfaches Verb verwenden. | Perego EASIT |

## 4. Regelset – Einfache Sprache (Linguaggio chiaro, Niveau B1)

### Unterschiede zur Leichten Sprache
| Kriterium | Lingua facile (A1–A2) | Linguaggio chiaro (B1) |
|---|---|---|
| Satzlänge | Sehr kurz (8–12 Wörter) | Kürzer als Standard (max. 20 Wörter) |
| Nebensätze | Keine | Max. 1 erlaubt |
| Congiuntivo | Nie | Sparsam |
| Fachbegriffe | Vermeiden oder erklären | Erklären |
| 1 Satz pro Zeile | Ja | Nein |
| Bilder | Empfohlen | Optional |
| Prüfgruppe | Empfohlen | Nicht obligatorisch |

### Wortschatz
- Gebräuchliche Wörter bevorzugen.
- Fachbegriffe erklären.
- Konkret statt abstrakt.
- Keine unnötigen Fremdwörter.

### Satzbau
- Kürzere Sätze als im Standarditalienisch.
- Aktiv bevorzugen.
- Indikativ bevorzugen.
- Logische Satzreihenfolge (Subjekt – Verb – Objekt).

### Textstruktur
- Klar gegliedert mit Überschriften.
- Absätze und Aufzählungen.
- Linksbündig.

## 5. Qualitätscheckliste

1. Enthält jeder Satz nur eine Aussage?
2. Beginnt jeder Satz auf einer neuen Zeile (Lingua facile)?
3. Steht alles im Indikativ (kein Congiuntivo)?
4. Sind alle Sätze im Aktiv formuliert (kein Passiv)?
5. Werden nur gebräuchliche, konkrete Wörter verwendet?
6. Gibt es keine Lateinismen, Fremdwörter oder Fachbegriffe ohne Erklärung?
7. Sind Euphemismen durch direkte Wörter ersetzt?
8. Sind Periphrasen durch einfache Verben ersetzt?
9. Wird dasselbe Wort für dieselbe Sache verwendet?
10. Gibt es keine doppelte Verneinung?
11. Wird keine «d eufonica» verwendet?
12. Gibt es keine unpersönlichen Formen?
13. Sind Zahlen als Ziffern geschrieben?
14. Sind Daten vollständig ausgeschrieben?
15. Ist die Schrift serifenlos, mindestens 14 pt?
16. Ist der Text linksbündig?
17. Gibt es klare Überschriften und Zwischentitel?
18. Sind Bilder/Piktogramme eingesetzt (erwachsenengerecht)?
19. Wurde der Text von einer Person der Zielgruppe geprüft?
20. Ist der Text auf die Kerninformationen beschränkt?

## 6. Beispiele

### 6.1 Schulkontext

**Originaltext:**
«L'iscrizione scolastica del bambino è obbligatoria a partire dal compimento del quarto anno di età entro il 31 luglio dell'anno scolastico di riferimento, conformemente alla legge sull'istruzione pubblica.»

**Einfache Sprache (B1):**
«Vostro figlio deve andare a scuola dall'età di 4 anni. Deve compiere 4 anni prima del 1° agosto. Questo è scritto nella legge sulla scuola.»

**Leichte Sprache (A2):**
«Vostro figlio ha 4 anni?  
Allora deve andare a scuola.  
Deve avere 4 anni prima del 1° agosto.  
Questa è una regola importante.  
È scritto nella legge.»

### 6.2 Behördenkontext

**Originaltext:**
«I beneficiari di prestazioni complementari sono tenuti a comunicare senza indugio qualsiasi cambiamento della propria situazione personale o finanziaria all'organo competente.»

**Einfache Sprache (B1):**
«Ricevete prestazioni complementari? Se qualcosa cambia nella vostra vita o nel vostro reddito, dovete comunicarlo subito all'ufficio responsabile.»

**Leichte Sprache (A2):**
«Voi ricevete soldi dallo Stato.  
Questi soldi si chiamano prestazioni complementari.  
Qualcosa cambia nella vostra vita?  
Per esempio: cambiate casa.  
Oppure: ricevete più soldi.  
Allora dovete dirlo all'ufficio.  
Dovete dirlo subito.»

### 6.3 Alltag

**Originaltext:**
«In caso di malfunzionamento dell'ascensore, si prega di utilizzare le scale di emergenza situate in fondo al corridoio e di contattare il servizio di manutenzione al numero indicato.»

**Einfache Sprache (B1):**
«L'ascensore non funziona? Usate le scale di emergenza. Sono in fondo al corridoio. Chiamate il servizio di riparazione. Il numero è indicato.»

**Leichte Sprache (A2):**
«L'ascensore non funziona.  
Prendete le scale.  
Le scale sono in fondo al corridoio.  
Chiamate questo numero: 091 123 45 67.  
Qualcuno vi aiuterà.»

## 7. Prompt-Baustein für GPT

```
Sei un esperto di comunicazione accessibile. Riformuli testi in «lingua facile» (livello A1–A2 del QCER) o in «linguaggio chiaro» (livello B1).

### Regole Lingua facile (A1–A2):
- 1 idea per frase. 1 frase per riga.
- Frasi brevi (max. 12 parole).
- Solo indicativo. Solo forma attiva. Solo frasi affermative.
- Vocabolario semplice, concreto, quotidiano. Niente gergo o abbreviazioni.
- Spiegare subito le parole difficili.
- Niente metafore, eufemismi, ironia.
- Sempre la stessa parola per la stessa cosa.
- Rivolgersi direttamente al lettore (voi).
- Niente congiuntivo. Niente passivo. Niente doppia negazione.
- Niente «d eufonica» (ed, od). Niente participio presente in subordinate.
- Niente forme impersonali. Numeri in cifre. Date complete.

### Regole Linguaggio chiaro (B1):
- Frasi di 15–20 parole max. 1 subordinata tollerata.
- Vocabolario corrente. Spiegare i termini tecnici.
- Forma attiva. Indicativo. Indirizzo diretto.
- Testo strutturato con titoli, sottotitoli, elenchi.

### Procedura:
1. Identificare il livello target (A1/A2 o B1).
2. Estrarre le informazioni essenziali.
3. Riformulare secondo le regole.
4. Verificare ogni frase con la checklist.

### Auto-verifica:
- Ogni frase contiene una sola idea?
- Il vocabolario è semplice e concreto?
- Il testo è all'indicativo, all'attivo, affermativo?
- Il testo è logicamente strutturato?
```
